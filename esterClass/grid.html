        .container {
            display: grid;
            gap: 30px;
            grid-template-columns: 100px 300px 500px;
        }
        
        YOu can give them width using percentage- here is how
        .container {
            display: grid;
            gap: 30px;
            grid-template-columns: 10% 30% 50%;
        }

        Alternatively you can repeat the smae with across the nunmber of boxes , let say you have a lot of boxes and you do not want to style one by one
        .container {
            display: grid;
            gap: 30px;
            grid-template-columns: repeat(3, 1fr);
          }

          Do you know you can specify the fraction length of each boxes

        .container {
            display: grid;
            gap: 30px;
            grid-template-columns: 2fr 1fr 3fr;

Responsive Grids Row and columns

Lets break Down How to Give a column Gap and a Row Gap

      
          #T0 space out the boxes and give them sizes using PX
          .resp-gaps {
            display: grid;
            grid-template-columns: repeat(12, 1fr);
    
            grid-template-columns: repeat(12, minmax(100px, 1fr));
            column-gap: 10px;
            row-gap: 10px;
    
            grid-template-columns: repeat(
              auto-fit,
              minmax(100px, 1fr)
            ); /* to make it responsive use Autofit   */
          }
    
          @media screen and (min-width: 780px) {
            .resp-gaps {
              display: grid;
              grid-template-columns: repeat(12, 1fr);
    
              grid-template-columns: repeat(12, minmax(100px, 1fr));
              column-gap: 10px;
              row-gap: 10px;
    
              grid-template-columns: repeat(
                auto-fit,
                minmax(100px, 1fr)
              ); /* to make it responsive use Autofit   */
            }
    
            .resp-gaps div:nth-child(2) {
              grid-row-start: span 3;
    
              /* grid-column: span 3 / auto; */

            .resp-gaps div:nth-child(1) {
              grid-row-start: span 2;
